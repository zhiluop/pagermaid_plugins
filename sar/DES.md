# SAR - 贴纸自动回复插件

## 功能描述

在指定群组中，当有人用贴纸回复你的消息时，自动回复相同的贴纸。

## 使用方法

### 管理命令

- `,sar on` - 在当前群组启用功能
- `,sar off` - 在当前群组禁用功能
- `,sar set <群组ID>` - 启用指定群组的功能
- `,sar list` - 查看所有启用的群组
- `,sar stats` - 查看统计信息

### 使用示例

1. 在群组中启用功能：
   ```
   ,sar on
   ```

2. 启用指定群组：
   ```
   ,sar set -1001234567890
   ```

3. 查看已启用的群组：
   ```
   ,sar list
   ```

## 注意事项

- 此功能只在群组中生效
- 需要有人用贴纸**回复**你的消息才会触发
- 会回复完全相同的贴纸
