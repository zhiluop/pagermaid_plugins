# 自动抽奖插件 (LuckyDraw)

在指定群组中自动识别红包/抽奖活动并发送口令参与。

## 功能特点

- 支持多种口令格式识别
- 脚本检测规避（敏感词过滤）
- 随机延迟发送（0.5-2秒）
- 群组独立开关
- 统计信息

## 使用方法

1. 在群组中使用 `,ldraw on` 启用功能
2. 当群组中有人发起红包/抽奖活动时，插件会自动识别并发送口令

## 支持的口令格式

- `领取密令: xxx`
- `参与关键词：「xxx」`
- `发送 xxx 进行领取`
- `口令: xxx`

## 管理命令

- `,ldraw on` - 启用当前群组
- `,ldraw off` - 禁用当前群组
- `,ldraw set <群组ID>` - 手动添加群组
- `,ldraw list` - 查看已启用群组
- `,ldraw stats` - 查看统计
- `,ldraw test <文本>` - 测试口令提取
